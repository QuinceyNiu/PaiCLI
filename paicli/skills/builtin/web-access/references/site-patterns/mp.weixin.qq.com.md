# mp.weixin.qq.com

## 架构判断

微信公众号文章通常是 SPA/动态渲染，`web_fetch` 很容易拿不到完整正文，或者只得到脚本、空正文、环境校验内容。

## 成功路径

优先使用 Chrome DevTools MCP 打开页面，等待文章容器或标题出现后，用 `take_snapshot` 读取正文。必要时滚动页面触发懒加载，再重新 snapshot。

## 常见失败

- 正文为空：切浏览器，不要在 `web_fetch` 上反复尝试。
- 页面提示环境异常或频繁访问：降低操作频率，必要时请求用户手动打开或确认。
- 需要登录态：切 shared 模式后重试。
